# AI报销工具使用说明

## 快速开始

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动后端服务**
   ```bash
   python server.py
   ```
   或直接运行 `start.bat`

3. **打开工具页面**
   用浏览器打开 `ai-reimburse-tool.html`

## 使用步骤

### 步骤1：填写基本信息
- 报销事由：如"学术会议差旅费"
- 报销方式：选择现金报销/银行转账/公务卡
- 报销人信息：填写姓名、学号、手机号
- 活动时间：选择具体日期

### 步骤2：上传票据
工具提供三个独立的上传区域：

**🔵 发票上传**
- 点击"选择发票"或拖拽图片到蓝色区域
- 支持 JPG、PNG、PDF 等格式
- 系统自动识别：
  - 发票金额
  - 商户名称
  - 开票日期
  - 商品明细

**🟢 订单上传**
- 点击"选择订单"或拖拽图片到绿色区域
- 系统自动识别：
  - 商品名称和数量
  - 单价和总价
  - 下单时间

**🟣 支付记录上传**
- 点击"选择支付记录"或拖拽图片到紫色区域
- 系统自动识别：
  - 支付金额
  - 支付时间
  - 支付方式和收款方

### 步骤3：AI处理
点击"生成报销Word"按钮，系统会：
1. OCR识别所有上传的图片
2. 提取关键信息并结构化
3. 进行合规性检查

### 步骤4：查看结果
- **校验结果**：显示完整项、缺失项和风险项
- **票据明细**：按发票、订单、支付记录分组显示
- **总金额**：自动计算所有票据的总金额

### 步骤5：导出文档
点击"下载报销文档"即可生成：
- 标准Word格式的报销材料
- 包含封面、明细表、签领表

## 注意事项

1. **图片要求**
   - 确保图片清晰，文字可辨认
   - 避免反光和阴影
   - 建议使用原图，不要过度压缩

2. **文件限制**
   - 单个文件不超过5MB
   - 支持JPG、PNG、PDF格式
   - 建议文件名为中文或英文

3. **网络要求**
   - 确保能访问智谱AI服务
   - API已配置，无需额外设置

4. **常见问题**
   - 如果识别失败，可能是图片质量问题
   - 可以重新上传清晰的图片
   - 多张同类型票据会分别识别

## 技术支持

如有问题，请检查：
1. 后端服务是否正常启动（看到"Uvicorn running..."）
2. 浏览器控制台是否有错误
3. 图片格式是否正确
